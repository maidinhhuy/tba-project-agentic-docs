Khi nào hiện nút "Giao milestone"?

Project status = IN_DEVELOPMENT
Milestone status = ACTIVE
Milestone review_status = PENDING_DELIVERY
Milestone đã deliver hoặc đang chờ review → ẩn button

Trạng thái milestone trên row

PENDING_DELIVERY → hiện nút "Giao milestone" (teal)
PENDING_REVIEW → badge "Đang chờ review" (yellow)
APPROVED → badge "Đã chấp nhận" (green)
REVISION_REQUESTED → badge "Cần chỉnh sửa" (orange)
